第一步：使用font-face声明字体 
@font-face {font-family: 'myfont';
    src: url('qta.eot'); /* IE9*/
    src: url('qta.eot?#iefix') format('embedded-opentype'), /* IE6-IE8 */
    url('qta.woff') format('woff'), /* chrome、firefox */
    url('qta.ttf') format('truetype'), /* chrome、firefox、opera、Safari, Android, iOS 4.2+*/
    url('qta.svg#iconfont') format('svg'); /* iOS 4.1- */
}
第二步：定义使用myfont的样式 
.ui-icon{
    font-family:"myfont" !important;
    font-size:16px;font-style:normal;
}
